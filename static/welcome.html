One app surface, two ways to use it.

The hosted page and Chrome side panel share the same React build. The extension adds content scripts and a Manifest V3 service worker so the app can read supported job pages and request form-filling work.

1

Set up the profile

Add a resume, bio, AI key or credits, templates, and application settings in the web app or side panel.

2

Load the job context

Paste a description in the browser page, or use Load Job from the side panel while viewing a supported job site.

3

Tailor and apply

Generate documents in the app, then use Apply in the extension to fill supported forms and upload generated PDFs.

What it supports today.

Easy Job Apps keeps manual document work available on the web page, and adds page-aware automation when the Chrome extension has a content script on the active job tab.

Manual web workflow

Search or paste job descriptions, manage profile data, edit generated text, and create PDFs from the browser page.

Tailored documents

Use stored resume, cover-letter, bio, and template data to generate job-specific document versions.

Extension assist

Open the same app in Chrome's side panel, load job-page text, fill supported forms, and upload generated PDFs.
